# Icônes pour les notifications

Ce dossier contient les icônes utilisées par le système de notifications push :

- `message.png` - Icône pour les messages du chat général
- `dm.png` - Icône pour les messages privés
- `file.png` - Icône pour les fichiers partagés
- `task.png` - Icône pour les tâches assignées
- `system.png` - Icône pour les notifications système
- `badge.png` - Badge pour les notifications groupées

## Spécifications

- Format : PNG avec transparence
- Taille : 64x64 pixels (recommandé)
- Arrière-plan : Transparent
- Style : Moderne, simple, reconnaissable

## Fallback

Si ces icônes ne sont pas présentes, le système utilisera `/favicon.ico` par défaut.
