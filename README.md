# Pepite Mail — Prototype UI (MVP)

Prototype front-end cliquable de la boîte mail IA ultra-compacte.

Fonctionnalités incluses
- Vue 3 panneaux: Dossiers | Liste emails | Lecture/Résumé/Tâches.
- Focus "Boîte de réception": liste plein écran, zéro distraction.
- Densité ajustable: Compact / Dense / Ultra.
- Actions rapides sur chaque mail: Lire | Résumer | Répondre | Transférer.
- Priorité (P1/P2/P3) + badges Urgent/Important.
- Résumé IA (mock) + Tâches liées (mock).

Lancer en local
- npm install
- npm run dev
- http://localhost:3000

Prochaines étapes (après validation UI)
- Connexion Gmail/Outlook (OAuth).
- Génération de résumés via IA.
- Persistance et temps-réel (Supabase).
